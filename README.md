# His Soldier (그리스도의 군사) 🛡️

'수레바퀴의 삶(The Wheel Illustration)' 원리에 기초한 영적 훈련 기록 애플리케이션

---

## 💡 개발 동기 (Why 'His Soldier'?)

본 애플리케이션은 **디모데후서 2장 3절** ("너는 그리스도 예수의 좋은 병사로 나와 함께 고난을 받으라")의 말씀을 바탕으로, 그리스도의 군사된 성도가 매일의 영적 전투에서 승리하도록 돕기 위해 개발되었습니다.

건강한 영적 성장을 위해 반드시 필요한 4가지 핵심 요소를 **'말씀, 기도, 교제, 전도'** 로 정의하고, 이를 체계적으로 기록함으로써 삶의 중심에 예수 그리스도를 모시는 것을 목적으로 합니다.

---

## 📖 성경적 근거와 배경 (Biblical Foundation)

### 1. 말씀 (The Word)
말씀은 영혼의 양식이자 영적 전쟁의 유일한 공격 무기입니다.
- "주의 말씀은 내 발에 등이요 내 길에 빛이니이다" (시편 119:105)
- "구원의 투구와 성령의 검 곧 하나님의 말씀을 가지라" (에베소서 6:17)

### 2. 기도 (Prayer)
기도는 하나님과의 호흡이며, 우리의 무능함을 인정하고 하나님의 주권을 신뢰하는 행위입니다.
- "쉬지 말고 기도하라" (데살로니가전서 5:17)
- "아무 것도 염려하지 말고 다만 모든 일에 기도와 간구로… 하나님께 아뢰라" (빌립보서 4:6)

### 3. 교제 (Fellowship)
성도는 그리스도의 몸 된 교회의 지체로서 서로 연결되어 있을 때 건강하게 성장합니다.
- "서로 돌아보아 사랑과 선행을 격려하며 모이기를 폐하는 어떤 사람들의 습관과 같이 하지 말고…" (히브리서 10:24-25)

### 4. 전도 (Evangelism)
전도는 주님이 우리에게 주신 지상 최대의 명령이자 교회의 존재 이유입니다.
- "너는 말씀을 전파하라 때를 얻든지 못 얻든지 항상 힘쓰라" (디모데후서 4:2)
- "그러므로 너희는 가서 모든 민족을 제자로 삼아…" (마태복음 28:19)

## ✨ 주요 기능 (Key Features)

### 🏠 홈 (매일의 영적 훈련 기록)
- **4대 훈련 체크**: 말씀 / 기도 / 교제 / 전도 실천 여부 체크 및 메모 작성
- **성경 암송**: 오늘 외운 암송 구절을 홈 화면에서 바로 입력하고 확인
- **스마트 체크리스트 (To-Do)**: 영적 훈련 외에 일상 스케줄도 함께 관리. 캘린더로 과거/미래 날짜에 일정 예약 가능
- **자동 저장**: 별도 저장 버튼 없이 변경 즉시 로컬 DB에 동기화
- **알림 마커**: 달력에서 완료하지 않은 일정이 있는 날짜에 `○` 마커 표시

### 💙 그레이스 습관 (아침·저녁 기도 루틴) — *v1.3.0 신규*
매일 아침과 저녁, 2번의 짧은 기도 루틴으로 하루를 하나님과 함께 시작하고 마무리합니다.

| 루틴 | 내용 |
|------|------|
| **아침 기도** ☀️ | 10분 카운트다운 타이머. 평안한 마음으로 하루를 기도로 준비 |
| **저녁 기도** 🌙 | 3단계 가이드: ① 감사한 일 → ② 맡기는 걱정 → ③ 오늘의 기도 메모 |

- **연속 기록 (스트릭)**: 저녁 기도까지 완료하면 N일째 카운트가 올라감
- **오늘의 상태**: 아침/저녁 각각 완료 여부를 카드로 직관적으로 확인
- **알림 배지**: 06:00 아침 기도, 22:00 저녁 기도 알림 안내

### 🙏 기도수첩 (그리스도인의 기도 가이드)
빈 칸을 채워나가는 것만으로 체계적인 기도를 드릴 수 있는 가이드를 제공합니다.

- **날짜별 기도 명언**: 매일 다른 영적 명언이 상단에 노출
- **4개 기도 카테고리**:
  1. 찬양과 감사 (영혼 구원 감사 / 기도 응답 감사)
  2. 자복과 회개 (고범죄 자복 / 부지불식간의 죄 / 용서)
  3. 중보 기도 (교회 / 전도 대상자 / 가족 / 직분자)
  4. 간구 및 영적 일기 (MEMO: 오늘 받은 은혜와 묵상)

### 📊 통계 & 히스토리
- 훈련 달성도 차트 및 히트맵으로 성장 추이 한눈에 확인
- 과거 날짜 기록 수정 가능 (히스토리 탭 → 날짜 선택)

### 🌙 다크 모드
- 시스템 설정에 따라 자동 전환되는 고품질 다크/라이트 UI

## 🧠 아키텍처 및 내부 로직 (Architecture)

### State Management
- **Provider 패턴** 기반 MVVM 구조
- `FaithProvider`: 4대 훈련 기록 관리 (CRUD + 날짜 탐색)
- `GraceProvider`: 아침/저녁 루틴 완료 상태 및 스트릭 관리

### Data Persistence
- **SharedPreferences** 로컬 저장소 사용
- 모든 데이터 변경 시 즉시 JSON 직렬화 후 저장
- 날짜 변경(자정 경과) 시 오늘 상태 자동 초기화

### UI Architecture
- `CustomScrollView` 기반 충돌 없는 스크롤 최적화
- 하단 날짜를 SSOT(단일 진실 공급원)로 관리해 날짜 이동 시 즉각 상태 동기화
- 선택 날짜가 오늘이 아닌 경우, FaithProvider 전역 탐색으로 예약 일정 데이터 분기 처리

### 성경 암송 위젯
- `FaithRecord.bibleMemo` 필드에 통합 관리
- 포커스 해제 시 저장 — 잦은 I/O 방지 및 부드러운 타이핑 경험
